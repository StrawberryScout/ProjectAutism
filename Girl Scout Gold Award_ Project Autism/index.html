Cora Guerin is a Girl Scout Ambassador. She created this app as a project for her Gold Award, which is the highest award a Girl Scout can earn. In order to earn this award, one needs to complete a sustainable project that fixes a problem in the community. The problem Cora wanted to solve is that there aren't a lot of helpful resources for autistic people themselves, let alone teenagers.
